Introduction to browserify
==========================

####Setup

```
>npm install
>source env.sh
```


####Gulp tasks

 + `clean`: deletes the `dist` directory
 + `build`: builds the app using `js` and `html` tasks
  + `js`: browserify bundle
  + `html`: copy HTML files to `dist`
 + `watch`: watch HTML & JS files with watchify
